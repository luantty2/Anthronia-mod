# 額突魯披亞 / 厄本阿夷

A minimal Unciv civilization mod containing two custom civilizations.

Included:

- `jsons/Nations.json`
- `Images/NationIcons/額突魯披亞.png`
- `Images/NationIcons/厄本阿夷.png`
- atlas files required by the official example-based structure

Civilization setup:

- Name: `額突魯披亞`
- Leader: `Danrio A.`
- Colors:
  - `outerColor`: `[205, 98, 71]`
  - `innerColor`: `[21, 21, 21]`
- Cities:
  - `Klaudfield`

Second civilization:

- Name: `厄本阿夷`
- Leader: `Saman`
- Colors:
  - `outerColor`: `[21, 21, 21]`
  - `innerColor`: `[255, 255, 255]`
- Cities:
  - `Gepetti`

Design notes:

- No UA
- No UU
- No UB
- No diplomacy flavor text

Suggested test flow:

1. Upload this folder as a GitHub repository.
2. In Unciv, open `Mods`.
3. Use `Download mod from URL`.
4. Paste the repository URL and let Unciv install it through the supported path.
